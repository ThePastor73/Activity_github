Bulding HTML List

Embedded Style sheet
A Style sheet placed withing the <head> section of HTML document using the <style> tag. It applies styles only to that specific page.
External Style sheet
A separate .css file that stores all styles for a website. It is linked to one or more HTML documents using the <link> tag in the <head> section, promoting reuse and consistency.
Inline style sheet
A style appled directly to an individual HTML element using the Style attribute inside the opening tag. it onlyaffects that one element
